1. What is an operating system?

A. Hardware component
B. Software that manages hardware and software resources
C. Programming language
D. Database system

Answer: B
Why: An operating system acts as an interface between users and hardware. It manages processes, memory, files, and devices.

2. Which of the following is not a function of an OS?

A. Process management
B. Memory management
C. Compilation
D. File management

Answer: C
Why: Compilation is performed by compilers, not the OS. The OS focuses on resource management and system coordination.

B. Program file
C. Hardware unit
D. Memory block

Answer: A
