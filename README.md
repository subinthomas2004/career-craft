# Career Craft

CAREER CRAFT is an AI-powered placement preparation and career readiness platform designed to help students systematically prepare for campus placements and job interviews. The platform addresses the limitations of traditional placement preparation methods by providing automated, personalized, and data-driven learning experiences.

## Features

- **AI Mock Interviews**: Real-time feedback using speech-to-text analysis.
- **ATS-Optimized Resume Builder**: Create resumes that pass Applicant Tracking Systems.
- **Adaptive Tests**: Technical and aptitude tests that adjust to your performance.
- **Skill Gap Analysis**: Compare your skills with job requirements and get learning recommendations.
- **Performance Analytics**: Detailed dashboards tracking soft skills, coding performance, and more.
- **Real-time Notifications**: Updates on interviews, quizzes, and hiring alerts.

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Redux Toolkit
- Chart.js

### Backend
- Node.js
- Express.js
- MongoDB

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
