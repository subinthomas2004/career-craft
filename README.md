# Career Craft 🚀

**AI-Powered Career Preparation Platform**

Career Craft is a comprehensive platform designed to help students and job seekers master their placement journey. By leveraging cutting-edge AI, it provides realistic simulations and personalized feedback for every stage of the recruitment process.

## 🌟 Key Features

*   **🤖 AI Mock Interviews**: Realistic technical and HR interview simulations using local LLMs (Ollama) and Groq.
*   **📄 Smart Resume Analyzer**: Automated resume scoring and improvement suggestions.
*   **💻 Coding Arena**: Integrated code editor and compiler for practicing DSA and competitive programming.
*   **🗣️ AI Group Discussions**: 3D virtual environment for practicing group discussions with AI avatars.
*   **⚖️ Debate Room**: Interactive debate sessions to improve argumentation and confidence.
*   **📚 Aptitude & Technical Quizzes**: Extensive question banks for OS, DBMS, CN, and general aptitude.
*   **⌨️ Typing Tests**: Speed and accuracy training for coding tests.
*   **📊 Performance Analytics**: Detailed dashboards to track progress and readiness.

## 🛠️ Tech Stack

*   **Frontend**: React, TypeScript, Vite, Tailwind CSS, Shadcn UI
*   **Backend**: Node.js, Express, MongoDB
*   **AI Integration**: Ollama, Groq SDK, LangChain
*   **3D/Visuals**: Three.js, React Three Fiber
*   **Communication**: Socket.io, WebRTC (Simple Peer)

## 🚀 Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install` (root, frontend, and backend).
3.  Set up environment variables in `.env`.
4.  Run `npm run dev` to start the development servers.
